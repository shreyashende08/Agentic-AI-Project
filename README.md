# Agentic AI Project
**Author:** Shreya Shailendra Shende  
**Theme:** Autonomous Systems / Decision-Making AI

## Project Overview
This project demonstrates an Agentic AI system capable of autonomous task planning, reasoning, and execution with minimal human intervention. It simulates scenarios like smart city, healthcare, logistics, and energy management.

## Repository Structure
- `code/`      → Python scripts implementing Agentic AI
- `results/`   → Simulation logs and output charts
- `diagrams/`  → Architecture and flowcharts
- `data/`      → Sample datasets (optional)

## Getting Started
### Prerequisites
- Python 3.11+
- Required libraries: `numpy`, `pandas`, `matplotlib`

### Running Simulation
```bash
cd code
python main.py                # Default scenario: smart_city
python main.py --scenario healthcare
python main.py --scenario logistics
python main.py --scenario energy
